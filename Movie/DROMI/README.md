# DROMI 用テンプレート（第一部・予告編シリアス版）

iPadアプリ [DROMI](https://www.fenrir-inc.com/jp/dromi/) 向けの素材です。  
DROMI本体に「テンプレートファイルを読み込む」機能はないため、**画像をシーン背景として挿入**する運用です。

## フォルダ構成

| パス | 内容 |
|------|------|
| `templates/scene_blank_16x9.png` | 空の1シーン用レイアウト（何度でも複製用） |
| `shots/Part1_Trailer_Serious/C01.png` … `C21.png` | カットごとのメタ情報入り背景（21枚） |
| `part1_trailer_serious_shots.json` | ショットリスト（秒数・台詞・演出） |
| `Part1_Trailer_Serious_DROMI_ShotList.md` | 人間が読むショット表 |
| `generate_templates.py` | PNG再生成スクリプト |

## iPadへの取り込み

1. PCの `Movie/DROMI/` を iCloud Drive / Google Drive / AirDrop で iPadへ送る  
2. 「ファイル」アプリで PNG を写真に保存するか、DROMIの画像挿入から直接選択  
3. DROMIで **新規 → アスペクト比 16:9** を選ぶ  

## DROMIでの作業手順（推奨）

1. **新規プロジェクト** … 16:9、シーン長は「ビデオコンテ設定」でデフォルト3秒（C17だけ2秒など個別調整）
2. **シーン1** … 画像挿入 → `C01.png` を配置 → 背景レイヤーに固定（または最下層）  
3. **新規レイヤー** … 枠内に Apple Pencil で絵コンテを描く  
4. **シーンの長さ** … 下表の `duration_sec` に合わせてタイムライン上で伸縮  
5. **複製** … 似た構図はシーン複製（中央ボタン）で効率化  
6. **ストーリーボード表示** … 書き出し時にセリフ・コメントを追加可能（PDF共有向け）  
7. **書き出し** … 編集用は **DROMI形式**、提出用は **PDF（表形式）** または **MP4**

## PNGの再生成

```bash
cd Movie/DROMI
python generate_templates.py
```

`part1_trailer_serious_shots.json` を編集してから実行すると、ショットカードが更新されます。

## 作画メモ（シリアス版）

- 表情は抑制。目・手・呼吸で語る  
- 色は低彩度。夕焼けも「酸化した赤」  
- AR・UIは薄いノイズ程度。派手なホログラム禁止  
- 台詞はJSON／PNG下部に記載済み。ストーリーボード書き出し時にコピー可  

## 注意

- DROMIのオーディオは **1トラックのみ**。SE用に1本のMP3/WAVを編集して載せる  
- 縦動画（9:16）用にする場合は `generate_templates.py` の `W, H` を 1080×1920 に変更して再生成

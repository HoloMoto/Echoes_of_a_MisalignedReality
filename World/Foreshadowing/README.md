# 伏線管理フォルダについて

このフォルダは「アネスティシア・アウェアネス」ストーリーにおける伏線の管理と追跡を目的としています。

## フォルダの目的
- 物語全体に配置された伏線を体系的に管理する
- 伏線の配置と回収のタイミングを明確に記録する
- 伏線同士の関連性を視覚化する
- 読者が気づくポイントと真相解明のバランスを計画する

## 含まれるファイル
1. **ForeshadowingMap.md** - 伏線の全体マップと相互関連
2. **CoreMysteries.md** - 核心的な謎とそれに関連する伏線
3. **CharacterSecrets.md** - 登場人物ごとの秘密と伏線
4. **TechnologyHints.md** - 技術設定に関する伏線
5. **PlacementSchedule.md** - 章・話ごとの伏線配置計画

## 使い方
このフォルダの内容は、物語を執筆する際の指針として活用できます：
- 新しいエピソードを書く前に、配置すべき伏線を確認
- すでに配置した伏線の回収タイミングを計画
- 伏線が過度に複雑になっていないか確認
- 物語の終盤で未回収の伏線がないか確認

## 更新ガイドライン
- 新しい伏線を追加した場合は、該当するファイルを更新する
- 伏線を回収した場合は、その状態を「配置済み→回収済み」に更新
- 定期的に伏線の全体像を見直し、物語の流れと整合性を確保する

CONCEPT

クロノスゲート

クロノスをめぐる「入口」「接続点」「境界」を指す概念。現実、記憶、ネットワークが重なる場所を表す。

核心に触れる可能性があります

クロノス・タワーが町の制御系、千花の内側、IKAROS残存領域へ接続する場面を「門」として捉えるための言葉。作中の「クロノスの扉」「クロノスへ繋がる線」に近い意味を持ち、クロノス現象を理解するための鍵になります。

FACILITY

クロノス・タワー

瑠璃見のダム湖周辺に存在する巨大サーバー施設。町のインフラと深く結びついている。

核心に触れる可能性があります

水冷系や同期観測線を備え、町の制御、千花の生体反応、過去の実験記録が交差する中枢。後半では「クロノスを止める」ことが町と記憶の両方に関わる選択になります。

MANUAL

アナログハッチ

クロノスの水冷系に残された物理停止手段。デジタル制御外から介入するための手がかり。

核心に触れる可能性があります

荒木の告発記録に示された外側の手段。クロノス側の遠隔制御や改竄された図面を越えて、人の手で閉じるために残された出口です。

LINE

同期観測線

クロノス建設時に使われたとされる隠れた接続線。学校や町の古い設備に痕跡が残る。

核心に触れる可能性があります

旧体育館床下の管路などに残る物理的な線。箱庭の記憶とクロノス・タワーをつなぎ、千花の「ズレた時間」を現実側へ引き寄せる鍵になります。

TECH

サードアイ / スマートアイ

日常生活に浸透した空間コンピュータ。世代ごとに形状や入力方法が変化してきた。

詳しく読む

初期のスタンドアロン型から、スマートフォンの代替、脳波入力、コンタクト型へと発展したデバイス群。作中ではサードアイとも呼ばれ、千花が使用するものは脳波入力を扱う第五世代にあたる。

NETWORK

ネオユビキタスネットワーク

2073年の社会を支える情報基盤。人と街とデバイスを常時接続するネットワーク。

詳しく読む

視界、行動、行政サービス、生活インフラを横断的につなぐ仕組み。便利さの裏側で、現実の見え方や個人の情報管理にも深く関わっています。

INTERFACE

ユビキタスリンカー

脳波とデジタル情報を接続する、極めて特殊なブレインマシンインターフェイス。

核心に触れる可能性があります

量子コンピュータ級の演算能力を、人間が扱える情報量へ翻訳するための装置。最初の被験者は春風千花で、「ズレた時間」やカイの成立にも関わります。

CORE

ユビキタスコア

ユビキタスリンカーとの接続点となるブラックボックス部。個体認証にも関わる。

核心に触れる可能性があります

脳波パターンを用いた暗号化や増幅を担う中核部品。安全装置であると同時に、接続者をめぐる争奪や保護の理由にもなります。

AI

レッドボックス

人類未到達の技術を生み出しうる、超高度AIや量子コンピュータを指す呼称。

詳しく読む

日本ではIKAROSや神楽が該当する。軍事・政治・生活技術の均衡を左右する存在として扱われ、核兵器級の抑止力として警戒されます。

AI

IKAROS

作中の重要な超高度AI。ユビキタスリンカーの思想やカイの成立に関わる存在。

核心に触れる可能性があります

荒木紘一の構想を内側に抱え、疑似人格カイ・ローレンスを作成した領域に関わります。クロノス停止時には残存領域の扱いが大きな選択になります。

AI

神楽

スサノオ部隊の支援にも使われる超高度AI。装備制御や作戦支援に関わる。

詳しく読む

強化スーツf62などの高度装備をバックアップし、対レッドボックス戦でも重要な役割を持つAIです。

GEAR

強化スーツ

身体保護と身体拡張を組み合わせた戦闘装備。特殊部隊で運用される。

詳しく読む

AI支援による姿勢制御や換装式ユニットを備え、任務に応じて飛行・高機動・防護などを拡張する。第6世代以降のスマートアイとの併用が基本です。

GEAR

強化スーツf62

スサノオに先行配備された新型パワードスーツ。任務に応じて構成を変えられる。

詳しく読む

神楽のバックアップを受けて身体を拡張する試作装備。飛行ユニットや高高度狙撃など、三次元的な機動を想定しています。

GEAR

シールド

スサノオ隊に先行配備された特殊素材の防弾シールド。f62開発のプロトタイプでもある。

詳しく読む

多層フィルムと電子回路を内蔵し、ビーム給電による相転移で強度を上げる。自己再生ガラスの性質を持ち、継続使用を想定した装備です。

DEVICE

クアンタムセル

強化スーツなどに用いられる超高度AIの子機端末。同期によって演算能力を拡張する。

詳しく読む

20cm大のセルにナンバリングされ、センサーや演算補助として使われる。雷鳥に搭載された神楽のターミナルハブは、この発想へ通じるプロトタイプです。

SUIT

ヘパイストス

ユビキタスリンカーユーザー専用の高高度強襲用空中装備。

詳しく読む

思考操作により身体を動かさず操縦でき、分離後は浮遊支援テザードローンとしても運用できる。隠密偵察や強襲に向いた装備です。

SUIT

ヘパイストスII

ユビキタスリンカーユーザー以外にも扱えるよう改良されたヘパイストス。

詳しく読む

脳波コントロールで操作可能になり、ハードポイントも増設された一方、重量増加や出力低下が見られる改良型です。

AIRCRAFT

輸送機C-2改「雷鳥」

自衛隊C-2輸送機をベースにした特殊作戦用輸送機。空中指揮所としても機能する。

詳しく読む

13課やスサノオ部隊などが運用する輸送・指揮統制機。監視スクリーンや強化スーツのコントロールパネル、隊員の空中展開機能を備えます。

DEVICE

VA64

Vision Amplifier type64型スマートアイ。スサノオ隊の標準装備。

詳しく読む

第6世代スマートアイに相当するバイザー型デバイス。うなじ部の脳波集約補助デバイスで高度な入力を行います。

WEAPON

VSAR

13課の標準装備である実弾型ライフル。読みはヴァーサー。

詳しく読む

Variable Speed Assault Rifleの略。実弾を使用し、MDシリーズ同様にAIによる自動照準機能を備えます。

ORG

公安局 / 第13課

ユビキタスコアをめぐる捜査や回収に関わる国家側組織。作中の対立軸の一つ。

核心に触れる可能性があります

国内機密情報を扱う組織として、ユビキタスコアの共同捜査や技術独占に関わる。琴音との癒着やクロノス計画をめぐり、スサノオ隊と衝突します。

ORG

スサノオ部隊

ユビキタスコアの保護・国外流出防止などに関わる自衛隊側の特務部隊。

詳しく読む

強化スーツf62、VA64、雷鳥などを運用する特殊部隊。13課と対立し、千花の保護をめぐる作戦に関与します。

PERSON

荒木紘一

ユビキタスコアおよびユビキタスリンカーの開発者。脳科学者であり量子コンピュータ研究者。

核心に触れる可能性があります

クロノス計画と千花の過去に関わる研究者。彼の告発記録は、事故、救命、実験、アナログハッチの存在をつなぐ重要な証拠になります。

# AI_Temp フォルダについて

このフォルダは、ストーリー「アネスティシア・アウェアネス」の情報整理と構造改善のための提案を集めたものです。

## フォルダの目的
- 複雑な設定と時系列を持つ物語の情報をより整理する
- 読者の理解を助ける情報提示方法を提案する
- ストーリー構造の改善案を提示する

## 含まれるファイル
1. **WorldSettings.md** - 物語の舞台となる世界の設定を整理
2. **CharacterRelations.md** - 登場人物とその関係性の整理
3. **Timeline.md** - 2062年と2063年の出来事を時系列で整理
4. **StoryStructure.md** - 情報の出し方や物語構造の改善提案

## 使い方
このフォルダの内容は、物語を執筆・編集する際の参考資料として活用できます。
特に以下のような場面で役立ちます：

- 新しいエピソードを書く前の設定確認
- 伏線の配置と回収の計画
- 登場人物の一貫性の確保
- 読者にとってわかりやすい情報開示の順序決定

## 更新について
物語の発展に合わせて、このフォルダの内容も適宜更新することをお勧めします。
新しい設定や登場人物が追加された場合は、関連するファイルを更新してください。

## 関連フォルダ
World/Foreshadowing フォルダには、物語内の伏線管理に関するファイルが含まれています。
